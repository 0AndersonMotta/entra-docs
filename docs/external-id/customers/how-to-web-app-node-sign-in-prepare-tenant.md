# Sign in users in my own Node.js web application by using Microsoft Entra - Prepare your tenant

In this article, you learn how to sign in users in your own Node.js web application. This article uses [Microsoft Authentication Library (MSAL) for Node](https://github.com/AzureAD/microsoft-authentication-library-for-js/tree/dev/lib/msal-node) to simplify adding authentication an authorization to your node web application.
