# How to: Delete a group using the Azure Active Directory portal
If you incorrectly set the **Group type** to the wrong option, if you mistakenly created a group for your organization, or if you just don't need the group anymore, you can delete your group. 

If you need to create a new group, follow the steps in [How to create a basic group and add members](active-directory-groups-create-azure-portal.md).
