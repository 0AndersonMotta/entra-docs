# Microsoft CloudKnox Permissions Management - Tasks pane

This topic describes how to use the **Tasks** pane to view task information in Microsoft CloudKnox Permissions Management.

- To display the **Tasks** pane, select **Tasks** (the timer icon) in the upper right of the screen.

    The **Tasks** pane displays the following information:
    - **Active** - A list of active tasks.

         If there are no active tasks, the following message appears **There are no active tasks**.
    - **Completed** - A list of completed tasks.

         If there are no completed tasks, the following message appears **There are no recently completed tasks**.
