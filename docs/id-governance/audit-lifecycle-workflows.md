Workflows created using Lifecycle Workflows are able to have their information stored for auditing information. This workflow information, contained in the audit log, are important in getting the full picture of everything that happened in your tenant. This log contains information up to the last 30 days. For more information, see [Audit Logs](lifecycle-workflow-history.md).
